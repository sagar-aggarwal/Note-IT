Note IT is a simple note taking application that allows you to create text notes, lists, and more with an interactive user interface.

##Features
##1.Notes
Different Kind of Notes such as text notes,lists and more can be recorded in the internal storage of the phone.All the notes can be viewed in either Grid View or List View Depending upon User's Taste.
